h2. Comptes Riseup Red

* *Compte Riseup Red*:  Ce compte est utilisé pour accéder au courriel (email), VPN, et chat.  On peut régler les paramètres en se pointant à https://account.riseup.net.
* *Mots de passe pour chaque service*:  Il est possible de configurer des mots de passe distincts pour VPN et chat en se rendant à https://account.riseup.net.  À titre d'exemple, c'est utile si on veut se servir du VPN sur son mobile sans devoir y laisser son mot de passe pour le courriel.

h2. Comptes Riseup Black

* *Compte Riseup Black*: Ce compte est utilisé pour l'accès au service VPN Black.  On peut changer le réglage des paramètres en se pointant à https://black.riseup.net.

h2. Autres comptes

* *Compte de liste de courriel*:  Si vous est abonné-e à une ou plusieurs listes à https://lists.riseup.net, vous avez un compte de liste de courriel.  On peut changer le réglage des paramètres en se pointant à https://lists.riseup.net.
* *Compte de groupe*:  Si vous utilisez https://we.riseup.net, votre compte pour ce site Web est indépendant de votre compte Riseup primaire comme de votre compte de liste de courriel.  Dans le futur, nous allons fusionner les comptes de groupe avec les comptes primaires.
